
# Redux Shopping cart

- From the JSON data i create a cart page with per-item price, per-item quantity, Total quantity and total amount
- It Increase and Decrease as per unit Quantity which automatically update the total Quantity and Amount.



## Features

- Product display with details and images
- Cart functionality: add and remove products, adjust quantities
- Total price calculation based on cart contents
